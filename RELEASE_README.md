# RecipeStudio — запуск релизной версии

Этот файл предназначен для пользователя релизного архива.

## Что внутри архива
- `RecipeStudio.Desktop.exe` или `RecipeStudio.Desktop` — готовое self-contained приложение для вашей ОС.
- `README.md` — эта инструкция.

## Как запустить
1. Распакуйте архив в любую папку.
2. На Windows запустите `RecipeStudio.Desktop.exe`.
3. На Linux запустите `./RecipeStudio.Desktop`.
4. Пользовательские файлы будут созданы в системных каталогах профиля, а не рядом с приложением.

## Установка на Debian/Astra Linux
1. Возьмите файл вида `recipestudio_<version>_amd64.deb`.
2. Установите его командой `sudo apt install ./recipestudio_<version>_amd64.deb`.
3. После установки приложение можно запускать как `recipestudio`.

## Базовый сценарий работы
1. Создайте новый рецепт или откройте существующий.
2. Отредактируйте точки и параметры.
3. Нажмите «Пересчитать».
4. Проверьте траекторию в визуализации.
5. Сохраните результат или экспортируйте в Excel.

## Важно
- Сборки различаются по ОС и архитектуре (`win-x64`, `win-x86`, `win-arm64`, `linux-x64`).
- Для Linux нужна графическая desktop-среда с OpenGL/X11 и утилита `xdg-open`.
- Для первичной ручной проверки на Astra Linux используйте чек-лист в `docs/astra-linux-smoke-checklist.md`.
