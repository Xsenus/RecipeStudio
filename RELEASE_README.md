# RecipeStudio — запуск релизной версии

Этот файл предназначен для пользователя релизного архива.

## Что внутри архива
- `RecipeStudio.Desktop.exe` — готовое приложение (Windows, self-contained).
- `README.md` — эта инструкция.

## Как запустить
1. Распакуйте архив в любую папку.
2. Запустите `RecipeStudio.Desktop.exe`.
3. При первом запуске рядом с приложением будут созданы рабочие файлы (например, БД).

## Базовый сценарий работы
1. Создайте новый рецепт или откройте существующий.
2. Отредактируйте точки и параметры.
3. Нажмите «Пересчитать».
4. Проверьте траекторию в визуализации.
5. Сохраните результат или экспортируйте в Excel.

## Важно
- Сборки различаются по архитектуре (`win-x64`, `win-x86`, `win-arm64`).
- Используйте архив, соответствующий вашей системе Windows.
