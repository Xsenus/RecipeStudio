# RecipeStudio

Десктоп-приложение на **Avalonia + .NET 8** для работы с технологическими рецептами: хранение, редактирование, расчет параметров и базовая симуляция траектории.

## Что обновлено

- Главная страница (Dashboard) переделана под dark industrial UI:
  - верхняя навигационная панель в стиле панели управления;
  - большой заголовок «ПАНЕЛЬ УПРАВЛЕНИЯ»;
  - блоки статуса (количество рецептов, состояние системы, активность);
  - карточки доступных рецептов и быстрый сценарий «Создать новый».
- Обновлена общая визуальная тема приложения: цвета, типографика, кнопки, статус-плашки.

## Возможности

- **Dashboard**
  - просмотр списка рецептов из SQLite;
  - создание нового рецепта;
  - открытие и удаление рецептов.
- **Редактор**
  - таблица точек рецепта;
  - пересчет расчетных полей;
  - график траектории и базовая симуляция.
- **Настройки**
  - директория хранения данных;
  - рабочие константы;
  - параметры визуализации.

## Технологии

- .NET 8
- Avalonia UI
- SQLite

## Запуск

```bash
cd src/RecipeStudio.Desktop
dotnet restore
dotnet run
```

## Публикация (один исполняемый файл)

Теперь проект настроен так, что `dotnet publish` по умолчанию собирает **self-contained single-file** сборку (без необходимости отдельно устанавливать .NET Runtime на целевой машине).

```bash
cd src/RecipeStudio.Desktop
dotnet publish -c Release
```

Готовый файл будет в каталоге:

`bin/Release/net8.0/win-x64/publish/`

## Структура проекта

- `src/RecipeStudio.Desktop/Views` — окна и страницы UI.
- `src/RecipeStudio.Desktop/ViewModels` — логика представлений (MVVM).
- `src/RecipeStudio.Desktop/Services` — работа с рецептами, БД и расчетами.
- `src/RecipeStudio.Desktop/Styles` — глобальная тема и стили приложения.

## Данные

- Основное хранение: SQLite (`recipes.sqlite`).
- Импорт/экспорт: Excel (`.xlsx`) и совместимость с CSV/TSV.
